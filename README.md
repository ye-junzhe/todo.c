# todo.c

A simple but useful multithreading CLI todo app

### Install

```
sh build.sh
./todo
```

### Keybindings

```
j => Move down
k => Move up
c => Create todo
x => Done todo
r => Rename todo
s => Refresh
h => Help
```

> Logging messages will clear themselves after 5 seconds, achieved by applying multithreading
